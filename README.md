# Exemple de police personnalisée dans CSS

1. ouvrir FontForge (créer un nouveau projet, ou ouvrir fichier .sfd)
2. **dessiner**
3. **importer** (SVG, etc.)
4. **exporter** (.ttf, etc.)
5. **utiliser** dans CSS !
6. exécuter `vite` dans le terminal et ouvrir dans un navigateur internet http://localhost:5173/
